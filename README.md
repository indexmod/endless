# 🟢 ENDLESS

Endless — это минималистичная система публикации и управления контентом, построенная как **живой поток записей**, а не как классическая CMS.

Здесь нет сложных ролей, логинов или админ-панелей в привычном смысле. Вместо этого система разделена на два простых интерфейса: создание и модерацию.

---

# 🌐 КАК ЭТО УСТРОЕНО

Endless работает как единый поток данных:

- все посты хранятся в облаке (KV-хранилище)
- каждый пост — объект с изображением и текстом
- изменения мгновенно отражаются в ленте

Система обновляется через API и не требует пересборки или ручного деплоя при работе с контентом.

---

# ✍️ ПУБЛИКАЦИЯ

Интерфейс публикации предназначен для добавления новых записей в систему.

Пользователь:
- вставляет ссылку на изображение
- пишет текст поста
- видит счётчик символов (лимит 500)
- нажимает **send**

После отправки:
- пост сохраняется в облаке
- пользователь перенаправляется на главную ленту

Это быстрый “потоковый” способ публикации без лишних шагов.

---

# 🧹 МОДЕРАЦИЯ

Интерфейс модерации — это упрощённый список всех постов системы.

Он предназначен только для одного действия:
- удаление записей

Каждый пост отображается как компактный элемент с текстовым превью и кнопкой **delete**.

После удаления:
- запись исчезает из облака
- изменения сразу отражаются в общей ленте

---

# 🧭 ГЛАВНАЯ ЛЕНТА

Главная страница Endless — это непрерывный поток контента.

Особенности:
- изображения и текст отображаются как карточки
- редактирование текста происходит прямо в ленте
- изменения автоматически сохраняются в облаке
- локальный backup защищает от потери данных

---

# 💾 ХРАНЕНИЕ ДАННЫХ

Все данные хранятся в едином облачном ключе:

- `feed` — массив всех постов

Каждый пост содержит:
- уникальный ID
- изображение (URL)
- текст

---

# ⚙️ ФИЛОСОФИЯ СИСТЕМЫ

Endless — это:

> “минимальная CMS без CMS”

Здесь:
- нет аккаунтов
- нет ролей
- нет лишней сложности

Только поток контента и простые операции:
- создать
- изменить
- удалить

---

# 🚀 КАК ЭТИМ ПОЛЬЗОВАТЬСЯ

## Создание поста
1. открыть админку
2. добавить изображение
3. написать текст
4. нажать **send**

## Редактирование
- редактировать текст прямо в ленте
- изменения сохраняются автоматически

## Модерация
- открыть moderator-интерфейс
- удалить ненужные посты

---

# 🟡 ИДЕЯ

Endless — это не платформа, а **поток данных**.

Контент здесь не “управляется”, а просто существует и изменяется в реальном времени.
